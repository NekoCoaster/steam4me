NAL & NBH Passenger Cars (VR narrow gauge)

3d models original design and textures by Craig Walton

Overview

Package contains sixteen carriages as operating today on the Puffing Billy line between Belgrave and Gembrook.

  • 1NBH, 2NBH, 3NBH, 7NBH, 15NBH, 16NBH, 19NBH, 20NBH, 22NBH (Excursion cars)
  • 11NBH (Excursion car & Brakevan)
  • 51NBH, 52NBH (Excursion car for wheelchairs)
  • 1NAL- Mount Lyell (first class saloon car)
  • 2NAL - Rinadeena (first class saloon car)
  • 3NAL - Teepookana (first class saloon car)
  • 4NAL - Dubbil Barrill (first class saloon car)

Sound Sets For MSTS Rolling Stock

Overview

Specific sound set files (usually with an installer) are required by some models located on this website.  While these are fully equipped packages, you generally only need to download and install these once (they will generally install into a "common.sound" or "common.snd" folder in your TRAINSET folder) and the various locomotives and vehicles will then call the sounds as required.

All are hosted here with the permission of the original designers.

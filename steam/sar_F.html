F class 4-6-2T Steam Lococmotive (South Australian Railways)

original 3D model and textures by Joseph Spinella
Physics, sound set integration by team-ALCO

Overview

Seven locomotives in package:

SAR F class 4-6-2T steam locomotives: road numbers 170, 171, 173, 243, 252, 253, 255

Versions include:

  • player engines;
  • helper engines (for use in player trains);
  • traffic (AI) engines

Player and AI versions of each locomotive are included. Release3 physics (double-couplers, buffers, braking, steaming) by Ian Bowles (20-Jun-2006).

The crew provided courtesy of Paul Gausden: see his web site at http://www31.brinkster.com/decapod/

This locomotive has been designed for use with the LightFx lighting enhancement. The installer will install this enhancement for you. Click on this image for more details.

Tasks Required

  1. Run the installer.   two light engine consists have been created and will be installed for you.
     
  2. Run the "2-cylinder sound set" installer available from the Sounds page.
     
  3. You will need to create a consist to use these locos in other trains.  Click here for a tutorial on building up your own consists. Note that this loco has chain couplers rather than automatic couplers.

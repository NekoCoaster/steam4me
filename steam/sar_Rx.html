Rx class 4-6-0 Steam Locomotive (South Australian Railways)

original 3D model and textures by Matt Austin
Physics, sound set integration by team-ALCO

Overview

Eight locomotives in package, four player and four AI locomotives:

  • SAR 4-6-0 steam locomotive Rx 93 (as preserved)
  • SAR 4-6-0 steam locomotive Rx 207
  • SAR 4-6-0 steam locomotive Rx 224
  • SAR 4-6-0 steam locomotive Rx 227

The crew provided courtesy of Paul Gausden: see his web site at http://www31.brinkster.com/decapod/

This locomotive has been designed for use with the LightFx lighting enhancement. The installer will install this enhancement for you. Click on this image for more details.

Additional Files Required

  1. This locomotive requires the 2-cylinder sound set files by Stuart Williamson for its sound.
     
  2. This locomotive contains sounds aliased to default Scotsman and 380 locomotives.
  3. This locomotive requires that the Victorian Railways' D3 locomotive be installed as this model utilises its cabview and sound files.
     

 Return to top of page

Tasks Required

  1. Run the installer.   Light engine consists for Rx 093 and Rx 227 have been created and will be installed for you.
     
  2. Run the "2-cylinder sound set" installer available from the Sounds page.
     
  3. You will need to create a consist to use these locos in other trains.  Click here for a tutorial on building up your own consists.
  4. Note: version 1.4 (released 19-05-2006):
    1. AI locomotives added;
    2. fine-tuning ENGine files;
    3. corrected numberplate ACE files,
    4. crew added: crew shapes and textures by Paul Gausden.

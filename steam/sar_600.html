600 class 4-6-2 Steam Locomotive
(South Australian Railways)

original 3D model and textures by Allan Lownsborough

Overview

Five locomotives in package:

  • SAR steam locomotive # 600, 604, 606, 607, 609

Player and AI versions of each locomotive are included.

Release3 physics (double-couplers, buffers, braking, steaming) by Ian Bowles (team-ALCO).

The crew provided courtesy of Paul Gausden: see his web site at http://www31.brinkster.com/decapod/


SAR 600

This locomotive has been designed for use with the LightFx lighting enhancement. The installer will install this enhancement for you. Click on this image for more details.

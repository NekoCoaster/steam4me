South Australia - Adelaide, Hills and South-East (Version 2.0)
by Allan Lownsborough

Overview

Over 6,000 hours of work: 1,300km track, 1,200km road and over 1,400 custom items - Allan Lownsborough's epic route - "Adelaide, Hills and South-East" extends from Serviceton in Victoria to Adelaide and beyond to Gawler in the north, from Mount Gambier in the south-east to Outer Harbour, from branches such as Victor Harbour, Mount Pleasant and Willunga to Adelaide. This is an incredibly detailed and polished route.

Overland from the wharf at Murray Bridge

The route comprises the following lines:

  • Adelaide - Willunga;
  • Adelaide - Northfield;
  • Adelaide - Henley Beach;
  • Adelaide - Outer Harbour;
  • Adelaide - Two Wells;
  • Adelaide - Penfield;
  • Adelaide - Gawler;
  • Adelaide - Mount Pleasant;
  • Glenelg tram line;
  • Adelaide - Mount Barker Junction;
  • Sandergrove - Milang;
  • Mount Barker Junction - Victor Harbour;
  • Mount Barker Junction - Wolseley;
  • Wolseley - Mount Gambier;
  • Wolseley - Serviceton - Kaniva.

Tasks Required

  1. You must have all six default routes installed BEFORE running this installer. The routes just need to be present: they DO NOT need to be unstored if you use Trainstore, nor do you need to activate them if you use Route Riter's ON/OFF utility.
     
  2. You must have Xtracks 3.18 or higher installed.
     
  3. You must have Newroads 3.2 or higher installed.
     
  4. Run the installer by double-clicking on the EXE file you have saved to your hard disk. In the case of the multi-part installers, make sure that all eight parts are in the same folder, then simply double-click on the EXE file to commence installation. The route will occupy about 1.6G of hard disk space.
     
  5. Allow the batch file(s) to run to completion.
     
  6. The first time you run this route, there will be a considerable delay as MSTS "generates terrain buffers" - this is a one-off and subsequent use of this route will load normally.
     
  7. You will need to have SAR rolling stock installed for the installed activities to work. These are all listed in the readme that accompanies the route: you may click here to download and/or read the readme (in PDF format).
     
  8. Note that links may refer to the now defunct Railpage site, all files, including the required James Brook's files listed in "Additional Files Required" above are here on this new site: but of course you've already seen that, haven't you?

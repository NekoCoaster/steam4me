Via Metrica Installation Guide
(Version 2, includes tsection.dat build 44)
by "Sierra" (a lapsed MSTS user on trensim.com)

Overview

Via Metrica is a range of meter-gauge track consisting of straight, curved, bridge, tunnel track pieces, as well as switches (turnouts), crossovers and a turntable.

Via Metrica For the End-User

You've downloaded a route and it says you need Via Metrica tracks to get it operating correctly. This is the download for you.

Download and install this latest version, which will ensure trouble-free operation. This installer also installs the latest version of tsection.dat (don't worry if you don't understand that, just know that it's a good thing to do).

Download Options

  • Single-file installer

    You prefer the convenience, accuracy, speed and simplicity of an installer.
    Click on the link below to commence downloading Via Metrica.

    VMTrack_v2_by_Sierra_EndUser_MSTS1_Setup.exe
    1. Download this installer and save it to your hard drive.
       
    2. You should already have made a back-up copy of your "Train Simulator\GLOBAL" folder (simply click on "Global" in Explorer, press press Ctrl-C | Ctrl-V and you'll have a "Copy Of Global" folder - just in case).
       
    3. Simply double-click on the saved VMTrack_v2_by_Sierra_EndUser_MSTS1_Setup.exe file and follow the prompts. The installer will find your MSTS installation.
       
    4. That's it - you're done! Fire up MSTS and away you go on your new route.
    Notes on the installer:
    1. The installer is a Qsetup installer and the Via Metrica installer has been written by Yuri Sos with permission from designer "Sierra";
    2. It will try to find the MSTS folder from your registry - it reads only, it DOES NOT WRITE TO THE REGISTRY;
    3. this installer will check existing files on your hard drive: IT WILL NOT OVER-WRITE existing files on your system that have a newer date stamp.
  • ZIP file archive

    You prefer the tedium, fiddliness and control of a .ZIP file.
    Click on the link below to commence download Via Metrica in ZIP form.

    vmt.zip
    1. Download this archive and save it to your hard drive.
       
    2. You should already have made a back-up copy of your "Train Simulator\GLOBAL" folder (simply click on "Global" in Explorer, press press Ctrl-C | Ctrl-V and you'll have a "Copy Of Global" folder - just in case).
       
    3. Unzip the file into a folder named "VM Track" on your hard disk. Make sure you select "All Files" and "Use folder names" when extracting.
       
    4. Copy the contents of folder VM Track\GLOBAL\SHAPES to folder Train Simulator\GLOBAL\SHAPES, overwrite ALL files when asked.
       
    5. Copy the file VMTrack\GLOBAL\tsection.dat to folder Train Simulator\GLOBAL\. 
    6. You will get a question: "Do you want to replace ... last changed date something ... with the file ... last changed date something else".
       
    7. If the date of your file (the first one) is LATER than the date of my file (the second one) you should answer "No".
      If mine is the newest you should say "Yes".
      If you don't get the question you've done something wrong.
       
    8. That's it - you're done! Fire up MSTS and away you go on your new route.

If all you want to do is operate a route you've downloaded or installed, STOP HERE, you're done.

Installing Train Sim Modeler into 64-bit OS


by Gerald "Ged" Saunders

Overview

Installing TSM into 64-bit OS has always been a troublesome operation. However, the previously accepted mode of installing into a 32-bit OS before transferring into a 64-bit OS and then "fiddling" with the Registry is no longer necessary.

I came across the ReactOS site which provided the essential component to complete this task.

The following procedure has been tested on Windows 7 Home Premium 64-Bit, but will probably also apply to XP, Vista and Win 8.

 Return to top of page

1. The Procedure If You Have the Download Version

The download version AbacusTSM.exe (which will probably be v1.1), or the Help Update (AbacusTSMHelp.exe), or the Library (AbacusTSMLibrary.exe), all use a 16-bit installer.

The following procedure may be used to install them:

  1. Run the .exe file:
  2. and when you get the message box "Unsupported 16-Bit Application" displayed, DO NOT click OK!;
  3. Go to C:\Users\your username\AppData\Local\Temp;
    Find the tempoary folder ~EXB0000 folder that's just been created:
  4. In it you'll find a folder called DISK1
  5. Copy/move DISK1 to a safe place; if you have more than one .exe file to be installed, I suggest you rename it to reflect its contents; here I've copied it to c:\_temp:
  6. Now click OK on the "Unsupported 16-Bit Application...." window.......
  7. ..... and click OK on the resultant window:
  8. When this window appears, click Cancel:
  9. The Reactos website provided a link to download the Setup Engine executable directly from Nathan Lineback's web page at toastytech.com;


    10.  
  10. Open Is3Engine.zip and "drag-n-drop" setup32.exe into your DISK1 folder you've saved above:
  11. Right-click on setup32.exe and Run As Administrator and your TSM install will proceed as normal:

2. The Procedure If You Have the Install CDs

  1. Create a temporary folder on your hard drive called TSM;
  2. Create a folder within TSM called TSM-CD - ensure it is NOT read-only;
  3. Copy all the files off the TSM CD into folder TSM-CD;
  4. The Reactos website provided a link to download the Setup Engine executable directly from Nathan Lineback's web page at toastytech.com;


    5.  
  5. Open Is3Engine.zip and "drag-n-drop" setup32.exe into your TSM-CD folder you've saved above:
  6. Right-click on setup32.exe and Run As Administrator and your TSM install will proceed as normal:

Issues to be Aware Of

There are two, relatively small, things which don't work:

  1. when the installation is progressing, the message "Setup is creating icons" is displayed, but they aren't!;
  2. if you try to uninstall TSM or a part of TSM, it may not always work.

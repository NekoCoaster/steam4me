Introduction - Anything I need to know?

Generally speaking even though MSTS is now over 20 years old, it will install without difficulty on all versions of Windows, both 32- and 64-bit. I wrote the first version of this tutorial back in 2005 and MSTS is still alive and well, albeit a bit long in the (graphics) tooth.

BUT it appears that MSTS WILL NOT RUN (it will install) if your computer/laptop is running Windows 10 and has graphics cards made by ATI/AMD - ie Radeon. Laptops and desktops running Intel HD Integrated Graphics may also not run MSTS.

Laptops and desktops running nVidia graphics cards work ok.

Many laptops have switchable graphics options between Intel HD Graphics (less power consumption) and nVidia graphics. Ensure that you can switch from Intel -> nVidia before running MSTS.

Changing track textures in MSTS<
by Jim "Sniper" Ward

Overview

Just for fun, another BAT from my belfry.

This batch file is safe even for dummies, bulletproof error checking routine to make sure you don't copy over textures in one of the default routes, which are after all the source of the four default track textures.

 Return to top of page

How to use it, What it does

To use this batch file correctly, paste the tracktex.bat into the root of any ADD-ON route, double click on it, choose from one of the four track textures.

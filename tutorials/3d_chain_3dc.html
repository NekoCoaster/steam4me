Making A 3D Chain In 3D Canvas Pro v7 Or Higher
by Rick aka "thegrindre"

Overview

I've written this tutorial for 3D Canvas Pro (3DC) because I've had e-mails and PMs from a few modelers of our railroad forum boards asking me how I did it. It was done using the Piper plug-in Tool (Piper is a plug-in originally created by "Deanville" and is included in the Train Simulator Plug-in Pack (320k) on the Amabilis web site). It makes perfect elbows for grab irons, hand railings, chain and other curved items.

First let me say that I can't possibly teach you how to operate 3D Canvas Pro because even I don't know all the power this application has, but I will take you through each step, one by one, as I've done it to make a pretty good looking length of 3D chain for our railroad sims. I'm going to assume that you know some basic functions about the operation of 3DC because I'm only going to share with you how I make 3D chain links and not teach you how to 'run' this application.

All of my tutorials have always been a "hands-on","do-as-I-do" type of instruction with lots of pictures and is as simple as possible so if you follow my lead, and do as I do, you'll end up with a very realistic looking length of 3D chain.

This method has about the lowest poly count that I've been able to come up with and still have good looking 3D chain links.

Let's Get Started

Ready? OK, here we go...

Fire up 3DC then left-click the three axis movements, X, Y, Z (upper left top of tool bar). We don't want our model moving around on us unnecessarily while clicking on various aspects of it. This will constrain all movement of any part of the model including any faces and/or points. (Hint: This is a good practice to follow every time you load 3DC, btw. It should be default.)

Grab a cylinder primitive and drop it off the "playing field" in any black area. Doing it this way will automatically center our object for us after clicking OK to the pop up message. (Typical Drag & Drop action here).

We're now going to give it its basic shape. Left-click on the object to select it: a blue border will encircle it. Adjust the Longitude to 3 (lower left of interface in Object Layers) then hit ENTER.

We also need to resize it: right-click on the object, select Scale then left-click on To Size
Enter the following figures in the text fields: X = 1, Y = 1, Z = .5.
Left-click OK or hit ENTER. This will give us a more rounded look to the finished link.
(How do I know that, you might ask? Well I have been fooling around with this project for quite some time and found this worked best for me. You can try your own experiments and figures: you might find something you like better.)

We now have the basic shape we want.

The next part is going to get a bit tricky so follow along closely. There may be other ways to do this but this is how I do it.

Using The Piper Tool

The Piper Tool won't create the elbow in the proper direction for this project so we have to spin our object around 180 degrees. Left-click on the object to select it (if it isn't already selected) then left-click on the Shift Selection icon (lower left of interface in Object Operations). Below that, under Object Layers, enter 180 in the Rotate Y text field box then hit ENTER. Make sure all the other text fields are zeroed.

We now have our object resized and shaped, and are ready to make the elbow we're going to use for our 3D chain link using the Piper Tool.

Left-click on the Point and Face Selection Tool (upper right corner, second button from the top) then left-click the top face to select it: it'll turn blue. Now open the Piper Tool and enter the values in the text fields as shown in the image below. Make sure the Bend About X Axis radio button is also checked in that window (this is another area that you can experiment with, using your own values). (The hard part is figuring out length, width, curvature and all the math that goes into creating an accurate model such as this or hand railings or whatever. In this case, I've done all that for you.)

Note that there are now two objects, one being the elbow - that's the one we want. Left-click on the Component Selection Tool upper right corner of the interface, with the big black arrow on it) then left-click on that straight part. Press DELETE. We're only going to need the elbow to make our 3D chain link.

Use the left corner gizmo to roll the object over to look at the bottom. We're going to solidify our object because we need to 'fix' that missing face. Left-click on the elbow to select it (if it isn't already selected) then left-lick the Solidify Selection icon (lower left of interface in Object Operations). We now have a face we can extrude from: that's the next step.

Jump up to the Component Selection Tool (upper right corner of the interface with the big black arrow on it), left-click it to select our object again, left-click our object, then left-click on the Shift Selection icon and enter the following vlaues in the text fields: remember that we used it above to flip it around; we now have to flip it over. Rotate Z, enter 180 then press ENTER. Use the left corner gizmo to roll your model back to the upright position for better viewing of the next step. Left-click on the Mirror Object icon (you should know where all the icons are by now). Again, 3DC won't render our object properly so we have to flip it over.

Final Clean Up

Let's view it in wire-frame mode. It'll be much easier to clean it up this way. Let's also take a look at the poly count as it stands now. The next two images should help you with all of that (the pull-down menu is just to the left of center at the top of the interface in the Main toolbar):


Looking at the wire-frame view we can see a few items that can be eliminated. Those six lines and six points on the straight sections can surely be trimmed off. Let's do it, then look at the poly count.

Jump to the Point and Face Selection Tool (upper right corner), left-click then hover your mouse pointer around one of the lines at its center. It'll turn yellow when you catch it. Freeze your mouse then right-click to open that window, left-click, Trim. We just want the line! This can be a tricky process so watch what you're doing. If you accidentally catch the whole face and trim that off, then you'll have to jump to the Top Tool Bar to hit the Undo button to try it again. To make matters worse, you won't know it until you go back into Smooth mode view. So, be careful trimming lines and points.

Go ahead and Trim off all six lines and all 6 points IN THAT ORDER then check the poly count. Again trim the lines first then the points (trust me!). Here are before and after shots, respectively.

Poly count is now at 60: we've knocked off 12 polygons and 12 points. I don't think we can do any better then this and still have a good looking 3D chain link, in my opinion but you're welcome to try.

Go back to Smooth view and bucket fill a rusty texture on it. Now you're ready to assemble it to make yourself a length of chain but that's easy. Once you make two attached links, you can copy and paste to make 4 links then copy again to make 8 links, then again to make 16 then...

Got the idea? OK, good. Now scale it down to be proportionally accurate for the sim. At the beginning, I showed you how to Scale your model: this time left-click on By Amount, not Size:

This option will let you scale it by a percent such as 25% of the actual size you've built it in here.

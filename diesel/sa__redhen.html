Redhen 300 and 400 Class DMU Railcars (SA)

original 3d model design and textures by Alexander Jamieson
Physics, sound set integration by team-ALCO

Overview

Package contains:

  • 300 class Redhens: 300, 301, 302, 306, 311, 321, 325, 332, 352, 364, 366;
  • 400 class Redhens: 404, 410, 416, 431, 435, 466;
  • 800 class Trailer cars: 820, 825, 870, 875.

Forward and reversed models included. Consists for each destination (21 in all) include:

  • South Line:
    • Redhen {x} 300 820 301;
    • Redhen {x} 311 875 364;
    • Redhen {x} 431 435;
    • Redhen {x} 435 431 404;
    • Redhen {x} 404 311 875 364;
       
  • North Line:
    • Redhen {+} 301 820 311;
    • Redhen {+} 302 825 366;
    • Redhen {+} 306 870 325;
    • Redhen {+} 366 870 352;
    • Redhen {+} 466 306 870 325;
    • Redhen {+} 410 435 431 466;
       
  • Outer Harbour Line:
    • Redhen {||} 325 825 306;
    • Redhen {||} 364 820 300;
    • Redhen {||} 366 870 302;
    • Redhen {||} 404 416 364 820 300;
       
  • Marino Line:
    • Redhen {<>} 332 875 321;
    • Redhen {<>} 302 825 366;
    • Redhen {<>} 410;
       
  • Henley Beach:
    • Redhen {-} 352 825 332;
    • Redhen {-} 416 404;
       
  • Finsbury Line:
    • Redhen {|} 466.
       

Files contain release3 physics and friction and player- & AI-lighting, all by team-ALCO.

These vehicles have been designed for use with the LightFx lighting enhancement. The installer will install this enhancement for you. Click on this image for more details.

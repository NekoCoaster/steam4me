Xplorer DMU (Countrylink)

original 3d model design and textures by Brendan Robinson
Physics, sound set integration by team-ALCO

Overview

Package contains

  • Buffet Car with driver's cabin;
  • Intermediate car;
  • Disabled toilet car with driver's cabin.

These are made up into 2-, 3-, 4-, 5-, 6-, 7-car sets in both a "towards Sydney" (up) and "away from Sydney" (down) configuration.

These version 2 models contain upgraded textures, rewritten ENG files containing upgraded physics and player- & AI-lighting, all by team-ALCO. The TE shape file has been reversed for correct operation as a trailing unit.

Some shape files have been reversed for correct operation as trailing units, while careful integration of the various components have seen the download size drop from version 1's 15MB to 4.05MB.

Voith auto transmission sounds by Belphegor complete the package.

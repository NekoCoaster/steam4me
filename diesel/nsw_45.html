45 Class Diesel Locomotives (NSW)

original 3d model design and textures by Gary Ball
Physics, sound set integration by team-ALCO

Overview

Package contains nineteen models:

  • 4503, 4515, 4522, 4539 in NSWGR Indian Red livery;
  • 4502, 4508 in NSWGR "Reverse" livery;
  • 4506, 4509, 4511, 4529 in SRANSW "red terror" livery;
  • 4512, 4519, 4528, 4535 in SRANSW "candy" livery;
These locomotives have been designed for use with the LightFx lighting enhancement. The installer will install this enhancement for you. Click on this image for more details.

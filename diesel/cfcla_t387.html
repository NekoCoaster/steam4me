CFCLA T387 EMD Diesel (ex-Yorke Peninsula Railways)

original 3d model design and textures by James Brook
Textures by James Brook and Richard Lauder
Physics by Ian Bowles

Overview

Packages contain

  • CFCLA T387: long hood leading
  • CFCLA T387 reversed: short hood leading

This model contains the latest (release3) enhanced friction, braking and resistance parameters by Team-ALCO (Ian Bowles). The front coupler fix has been applied to this model.

Cabview files with correct colours have also been included. Ensure that James Brook's T class cabview is installed before this model.

These locomotives have been designed for use with the LightFx lighting enhancement. The installer will install this enhancement for you. Click on this image for more details.

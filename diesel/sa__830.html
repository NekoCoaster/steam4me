830 Class Goodwin-Alco Model DL531 Diesel (SA)

original 3d model design by Gary Ball
Physics, sound set integration by team-ALCO

Overview

Ten models in package:

  • 830, 836, 844, 858 in SAR "Mustard Pot" livery;
  • 859, 862 in ANR red and silver ;
  • 832, 835, 841, 864 in AN green & gold.

These models contain enhanced friction, braking and resistance parameters by team-ALCO, specifically by Ian Bowles.

These locomotives have been designed for use with the LightFx lighting enhancement. The installer will install this enhancement for you. Click on this image for more details.
